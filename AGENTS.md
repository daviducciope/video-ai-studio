# AGENTS.md

Indicazioni operative del repository:

- mantenere il progetto CLI-first e facilmente avviabile in locale;
- usare mock chiari e sostituibili quando il motore reale non e' disponibile;
- non hardcodare credenziali o path non necessari;
- aggiornare `docs/roadmap.md` quando una milestone cambia stato;
- preservare la separazione tra frontend, backend, worker e codice condiviso.
